Headless Press API

The Headless Press API is a RESTful backend service built with Node.js and PostgreSQL. It provides programmatic access to Articles, Topics, Users, and Comments, enabling social news platform functionality through standard HTTP methods.

Endpoints Available

  • GET /api/topics

    Retrieve a list of all topics.

    Example response:

    
{
  "topics": [
    {
      "slug": "coding",
      "description": "Code is love, code is life"
    },
    {
      "slug": "football",
      "description": "FOOTIE!"
    },
    {
      "slug": "cooking",
      "description": "Hey good looking, what you got cooking?"
    }
  ]
}

  • GET /api

    Serves a JSON representation of all available endpoints.

    Example response:

    
{
  "endpoints": {
    "GET /api": {
      "description": "serves up a json representation of all the available endpoints of the api"
    },
    "GET /api/topics": {
      "description": "serves an array of all topics",
      "queries": [],
      "exampleResponse": {
        "topics": [{ "slug": "football", "description": "Footie!" }]
      }
    }
  }
}

  • GET /api/articles

    Serves an array of all articles.
    Available Queries: author, topic, sort_by, order.

    Example response:

    
{
  "articles": [
    {
      "author": "weegembump",
      "title": "Seafood substitutions are increasing",
      "article_id": 1,
      "topic": "cooking",
      "created_at": "2018-05-30T15:59:13.341Z",
      "votes": 0,
      "article_img_url": "...",
      "comment_count": 6
    }
  ]
}

  • GET /api/articles/:article_id

    Serves a single article object by its ID.

    Example response:

    
{
  "article": {
    "article_id": 1,
    "title": "Living in the shadow of a great man",
    "topic": "mitch",
    "author": "butter_bridge",
    "body": "I find this existence challenging",
    "created_at": "2020-07-09T20:11:00.000Z",
    "votes": 100,
    "article_img_url": "...",
    "comment_count": 11
  }
}

  • PATCH /api/articles/:article_id

    Updates the vote count of an article.

    Example Request Body:

    
{
  "inc_votes": 1
}

    Example response:

    
{
  "article": {
    "article_id": 1,
    "title": "Living in the shadow of a great man",
    "votes": 101,
    ...
  }
}

  • GET /api/articles/:article_id/comments

    Serves an array of comments for the specified article.

    Example response:

    
{
  "comments": [
    {
      "comment_id": 1,
      "votes": 16,
      "created_at": "2018-05-30T15:59:13.341Z",
      "author": "butter_bridge",
      "body": "The beautiful thing about treasure is that it exists."
    }
  ]
}

  • POST /api/articles/:article_id/comments

    Adds a new comment to an article.

    Example Request Body:

    
{
  "username": "butter_bridge",
  "body": "I didn't know that!"
}

    Example response:

    
{
  "comment": {
    "comment_id": 19,
    "votes": 0,
    "created_at": "2020-09-19T23:10:59.341Z",
    "author": "butter_bridge",
    "body": "I didn't know that!"
  }
}

  • DELETE /api/comments/:comment_id

    Deletes a comment by its ID.

    Example response:

    Status 204: No Content

  • GET /api/users

    Serves an array of all users.

    Example response:

    
{
  "users": [
    {
      "username": "butter_bridge",
      "name": "jonny",
      "avatar_url": "https://www.healthytherapies.com/wp-content/uploads/2016/06/Lime3.jpg"
    }
  ]
}
